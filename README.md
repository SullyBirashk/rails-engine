* How to run the test suite
- run ```git clone git@github.com:SullyBirashk/rails-engine.git``` in your terminal
- run ```rails db{drop, create, migrate, seed}``` in your terminal
- run ```rails s``` in your terminal
- open postman and type in ```http://localhost:3000/api/v1/(your route here)```
- Watch as all the information flows in and you can access any chosen api!
- Here are a list of routes you could take
- ```get http://localhost:3000/api/v1/merchants```
- ```get http://localhost:3000/api/v1/merchants/:merchant_id```
- ```get http://localhost:3000/api/v1/merchants/:merchant_id/items```
- ```get http://localhost:3000/api/v1/items```
- ```get http://localhost:3000/api/v1/items/:item_id```
- ```get http://localhost:3000/api/v1/items```
- ```post http://localhost:3000/api/v1/items/:item_id```
